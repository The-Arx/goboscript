# Sound Blocks

### play sound () until done

```goboscript
play_sound_until_done "sound";
```

```_ {.scratchblocks}
play sound (sound v) until done
```

### start sound ()

```goboscript
start_sound "sound";
```

```_ {.scratchblocks}
start sound (sound v)
```

### stop all sounds

```goboscript
stop_all_sounds;
```

```_ {.scratchblocks}
stop all sounds
```
