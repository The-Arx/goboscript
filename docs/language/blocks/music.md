# Music Blocks

### rest for () beats

```goboscript
rest beats;
```

```_ {.scratchblocks}
rest for (beats) beats
```

### set tempo to ()

```goboscript
set_tempo tempo_;
```

```_ {.scratchblocks}
set tempo to (tempo_)
```

### change tempo by ()

```goboscript
change_tempo tempo_;
```

```_ {.scratchblocks}
change tempo by (tempo_)
```
